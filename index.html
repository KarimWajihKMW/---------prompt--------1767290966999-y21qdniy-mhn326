قسم Termux

المحاكي الطرفي الأقوى للأندرويد.

ما هو Termux؟

تطبيق مفتوح المصدر يوفر بيئة Linux كاملة على هاتفك الأندرويد دون الحاجة لترويت الجهاز. يمكنك من خلاله تثبيت لغات برمجة، أدوات اختراق، وسيرفرات.

التحميل الرسمي (تجنب جوجل بلاي لأنه غير محدث):

تحميل من F-Droid

1. تحديث الحزم (أساسي)

يجب تنفيذ هذا الأمر فور تثبيت التطبيق لتحديث المكتبات.

pkg update && pkg upgrade -y

2. منح صلاحيات التخزين

للسماح لـ Termux بالوصول لملفات الهاتف.

termux-setup-storage

3. تثبيت Git و Python

أهم الأدوات لتشغيل السكربتات.

pkg install git python python2 -y

قسم Kali Linux

نظام القراصنة والمحترفين.

على الكمبيوتر (PC)

يتم التثبيت كنظام أساسي أو وهمي (VirtualBox).

تحميل ISO الرسمي

على الهاتف (NetHunter)

نسخة مخصصة للأندرويد. تتطلب روت لكامل الصلاحيات، أو Termux لنسخة Rootless.

شرح NetHunter

تحديث النظام (Kali/Debian)

أمر التحديث الشامل للنظام.

sudo apt update && sudo apt full-upgrade -y

تثبيت الأدوات الشائعة

تثبيت الحزمة الكاملة لأهم 10 أدوات.

sudo apt install kali-tools-top10

أدوات الأمن السيبراني

أفضل 5 أدوات تعمل بدون روت (Rootless) وكيفية استخدامها.

1. Nmap (فحص الشبكات)

استطلاع

أداة قوية لاكتشاف الأجهزة المتصلة بالشبكة وفحص المنافذ المفتوحة.

التثبيت:

pkg install nmap -y

أمر التشغيل (فحص موقع):

nmap google.com

2. Sqlmap (حقن SQL)

Web Attack

أداة آلية لاكتشاف واستغلال ثغرات حقن قواعد البيانات (SQL Injection).

التثبيت (Git):

git clone --depth 1 https://github.com/sqlmapproject/sqlmap.git sqlmap-dev

3. Metasploit (مشروع الاستغلال)

Framework

إطار عمل ضخم لتطوير وتنفيذ الثغرات الأمنية (Exploits).

التثبيت (سكربت جاهز لـ Termux):

pkg install wget && wget https://raw.githubusercontent.com/gushmazuko/metasploit_in_termux/master/metasploit.sh && bash metasploit.sh

4. Hydra (تخمين كلمات المرور)

Brute Force

أسرع أداة لكسر كلمات المرور عبر الشبكة (SSH, FTP, etc).

التثبيت:

pkg install hydra -y

5. Sherlock (OSINT)

Information Gathering

البحث عن حسابات المستخدمين عبر منصات التواصل الاجتماعي.

التثبيت:

git clone https://github.com/sherlock-project/sherlock.git && cd sherlock && pip install -r requirements.txt
